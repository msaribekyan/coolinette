# Coolinette

## 42 code checker, which is cooler than Moulinette

Supports C00 and C01, for now.

## Setup
Clone the github repository to your home directory with the name "coolinette". Add the following alias to your bashrc/zshrc file:
```
alias coolinette="${HOME}/coolinette/coolinette"
```

## Usage
```
$ coolinette
```
