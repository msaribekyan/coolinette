# Coolinette

## 42 Piscine code evaluator, which is cooler than Moulinette

Supported project: C00, C01, C02 and C03.

## Setup
Clone the github repository to your home directory with the name "coolinette".
```
git clone https://github.com/msaribekyan/coolinette.git ~/coolinette
```
Add the following alias to your bashrc/zshrc file:
```
alias coolinette="${HOME}/coolinette/coolinette"
```

## Usage
Coolinette automatically detects project by directory name. Navigate into the project directory, and run:
```
$ coolinette
```

## Updating
During the piscine, several other projects will be added to Coolinette. To be up to date, navitage into the coolinette directory in your home directory and run:
```
git pull
```
